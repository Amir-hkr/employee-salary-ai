# 💼 Employee Salary Prediction ML Project

This project predicts employee salary based on demographic and professional features using Machine Learning.
---
<img src="test model.gif" width="900">

---
## 🚀 Project Goal
Build a regression model to estimate employee salary using features like age, experience, education, and gender.

---

## 📊 Dataset
- Source: Kaggle
- Size: ~323 cleaned records
- Features:
  - Age
  - Gender
  - Education Level
  - Years of Experience

---

## 🧠 Machine Learning Model
- Algorithm: Random Forest Regressor
- Performance:
  - R² Score: ~0.88+
  - MAE: ~11,000

---

## ⚙️ Workflow
1. Data Cleaning
2. Feature Engineering
3. Encoding Categorical Data
4. Model Training
5. Evaluation
6. Deployment (Streamlit App)
